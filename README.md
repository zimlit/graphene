## Getting Started

### Prerequisites

* golang 1.18 or later
* git

### Installation

1. clone the repo
``` sh
    git clone https://github.com/zimlit/graphene
```
2. build and install
``` sh
    cd graphene
    go install zimlit/graphene
```
3. make sure ~/go/bin is in your $PATH variable
